주요 기능

group

회원관리

회원 정보, 등록 현황, 출석 기록을 한눈에 파악하고 효율적으로 관리합니다.

  • check 회원 정보 통합 관리
  • check 출석 체크 자동화
  • check 회원권 만료 알림
calendar_month

PT관리

트레이너와 회원 간 PT 일정을 손쉽게 조율하고 관리합니다.

  • check 드래그 앤 드롭 스케줄링
  • check 자동 리마인드 알림
  • check PT 횟수 자동 차감
credit_card

자동결제

정기 결제와 미수금 관리를 자동화하여 매출 누락을 방지합니다.

  • check 정기 결제 자동 청구
  • check 미수금 알림 발송
  • check 결제 내역 자동 기록
dashboard

대시보드

센터 운영 현황을 실시간으로 파악하고 데이터 기반 의사결정을 지원합니다.

  • check 실시간 매출 현황
  • check 회원 통계 분석
  • check 트레이너별 실적 리포트
forum

업무게시판

트레이너 간 업무 공유와 공지사항을 효율적으로 관리합니다.

  • check 공지사항 일괄 전달
  • check 회원 인수인계 기록
  • check 업무 체크리스트
smart_toy

AI 챗봇

24시간 회원 문의에 자동으로 응대하여 업무 부담을 줄여줍니다.

  • check 24시간 자동 응답
  • check FAQ 자동 처리
  • check 상담 내역 자동 저장
