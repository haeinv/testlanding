## 1. Vercel CLI 설치 및 자동 배포 설정

### 문제
- npm 권한 오류로 Vercel CLI 설치 불가

### 해결 방법
```bash
# Homebrew로 Vercel CLI 설치
brew install vercel-cli

# 버전 확인
vercel --version

# Vercel 로그인 (브라우저 인증)
vercel login

# 프로젝트 연결
vercel link --yes

# 프로덕션 배포
vercel --prod --yes
```

### vercel.json 설정
프로젝트 루트에 `vercel.json` 파일 생성:
```json
{
  "version": 2,
  "buildCommand": null,
  "outputDirectory": ".",
  "routes": [
    { "handle": "filesystem" },
    { "src": "/(.*)", "dest": "/$1" }
  ]
}
```

**참고:** `outputDirectory`를 `.`로 설정해야 루트의 HTML 파일들이 제대로 배포됨

## 2. Google 로그인 오류 해결

### 문제
- Vercel 배포 후 Google 로그인 실패

### 해결 방법
Firebase Console에서 승인된 도메인 추가:

1. [Firebase Console](https://console.firebase.google.com) 접속
2. **Authentication** → **Settings** → **Authorized domains**
3. **Add domain** 클릭 후 추가:
   - `landing-blond-beta.vercel.app`
   - `vercel.app` (선택사항: 모든 Vercel 서브도메인 허용)

## 3. Particle Canvas 사이즈 확장

### 변경 파일
`index.html`

### 변경 전
```javascript
// Set canvas size
function resizeCanvas() {
    const container = document.getElementById('particle-hero');
    canvas.width = Math.min(1000, window.innerWidth - 32);
    canvas.height = Math.min(500, window.innerHeight * 0.6);
}
```

### 변경 후
```javascript
// Set canvas size
function resizeCanvas() {
    const container = document.getElementById('particle-hero');
    canvas.width = Math.min(1400, window.innerWidth - 32);
    canvas.height = Math.min(700, window.innerHeight * 0.75);
}
```

**변경 내용:**
- 최대 너비: 1000px → 1400px
- 최대 높이: 500px → 700px
- 화면 비율: 60% → 75%

## 5. 마이페이지 상세 페이지 추가

### 생성된 파일

| 파일명 | 용도 |
|--------|------|
| `mypage-purchase.html` | 구매내역 페이지 |
| `mypage-download.html` | 다운로드 페이지 |
| `mypage-support.html` | 고객지원 페이지 |
| `mypage-settings.html` | 계정설정 페이지 |

### 각 페이지 기능

#### 구매내역 (mypage-purchase.html)
- 총 결제금액, 결제건수, 현재 구독 상태, 다음 결제일 표시
- 검색 및 필터 (상태별, 기간별)
- 결제 내역 테이블 (주문번호, 상품명, 금액, 결제방법, 날짜, 상태, 영수증)
- Firebase Firestore에서 결제 데이터 조회

#### 다운로드 (mypage-download.html)
- 카테고리 탭 (전체, 소프트웨어, 문서, 리소스)
- 소프트웨어: Windows/macOS 설치 파일
- 문서: PDF 사용 설명서, 가이드북
- 리소스: 엑셀 템플릿, 마케팅 이미지

#### 고객지원 (mypage-support.html)
- 빠른 연락처 (전화, 이메일, 카카오톡)
- 자주 묻는 질문 (FAQ) 아코디언
- 1:1 문의 폼 (문의 유형, 제목, 내용, 첨부파일)
- 내 문의 내역 조회

#### 계정설정 (mypage-settings.html)
- 프로필 정보 수정 (이름, 연락처, 센터명, 프로필 이미지)
- 비밀번호 변경
- 알림 설정 (이메일, 마케팅, 결제 알림)
- 구독 관리
- 계정 삭제 (확인 모달)
